เกี่ยวกับโปรเจ็ค

เทมเพลต Django + Tailwind ที่ตั้งค่า Docker ให้พร้อมใช้งาน เหมาะสำหรับเริ่มต้นโปรเจ็คเว็บของคุณอย่างรวดเร็ว

กลับหน้าแรก เรียนรู้ Tailwind →

Stack

  • Django 5
  • Tailwind CSS 3
  • Docker + Compose
  • Whitenoise static

พร้อมเริ่มเร็ว

สั่ง `docker compose up --build` แล้วเริ่มพัฒนาได้ทันที มี watcher ของ Tailwind ให้พร้อม

โครงสร้างง่าย

เทมเพลตแบ่งส่วนด้วย `base.html` และหน้าเฉพาะใน `core/templates/core/`

ขยายได้คล่อง

เสริมแอป, เพิ่ม Static/Media, และปรับ Tailwind ได้ตามต้องการ

เริ่มปรับแต่ง

  • แก้ไฟล์ `templates/base.html` เพื่อทำ Header/Footer และ Navigation
  • แก้ CSS ที่ `assets/css/input.css` เพื่อเพิ่ม Components/Utilities
  • เพิ่ม Views/URLs ใน `core/views.py` และ `core/urls.py`
